    Introduction to The Linux Command Line (Bash Shell)

    Feb. 10, 10:00 am–noon • DCL L440

    When you open up a command line or need to execute a script, it's an alien world: man pages, terse and cryptic commands, unanticipated shortcuts and complications. We will review many of the most common commands necessary to effectively use the command line for running code and managing data, as well as scripting and more advanced topics.

    • Common commands
    • Environment variables
    • Scripting & program logic

    Introduction to Scientific Computing on a Cluster

    Feb. 17, 10:00 am–noon • DCL L440

    This is intended to give you a whirlwind tour of the issues underlying scientific computing, particularly as that is executed today on distributed clusters. What you should walk out of here with is at least a list of topics to dig deeper into, because semester-long courses could be written with the outline I am using to introduce scientific computing.

    • Scientific computing
    • Architecture
    • Performance
    • Running code (shells, job queues, scripts)
    • Programming, building, and debugging scientific codes
    • Numerical error

    Introduction to Illinois Campus Cluster

    Feb. 24, 10:00 am–noon • DCL L440

    While the prior workshop focuses on scientific cluster computing writ large, this workshop will focus more on the workflow, libraries, and environment of the Illinois Campus Cluster Program.

    • Background, design of CC
    • Access
    • Filesystem
    • Running code (Torque/PBS queueing)
    • Compilation, libraries
    • Making CC a bit friendlier to use
    • CC user resources

    Campus Cluster is now hosting walk-in help in 3101 NCSA, from 9 am to 5 pm, Mon through Fri.
